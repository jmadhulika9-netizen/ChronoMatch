# ChronoMatch

ChronoMatch is a luxury watch discovery platform that helps users find watches based on their style, budget, movement, and brand preferences.

## Features

- Personalized watch recommendations
- Match engine with fallback recommendations
- Watch collection management
- Watch personality insights
- Dark mode
- Responsive design
- Live deployment with Cloudflare Pages

## Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage
- GitHub
- Cloudflare Pages
